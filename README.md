# 🌤 Weather Dashboard

A modern Weather Dashboard built using Flask and OpenWeather API that provides real-time weather information, weather forecasts, and recent search history.

## 🚀 Features

* Real-Time Weather Data
* City Search Functionality
* Weather Icons
* Temperature Display
* Humidity Information
* Wind Speed Information
* 5-Day Forecast
* Recent Search History
* Clear Search History
* Responsive User Interface
* Error Handling

## 🛠 Tech Stack

* Python
* Flask
* HTML
* CSS
* OpenWeather API
* Requests Library

## ▶️ Installation

1. Clone the repository

2. Install dependencies

pip install flask requests

3. Run the application

python app.py

4. Open browser

http://127.0.0.1:5000
